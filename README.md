# 🌡️ Метеостанция на ESP8266 и DHT11

![ESP8266](https://img.shields.io/badge/ESP8266-NodeMCU-blue)
![DHT11](https://img.shields.io/badge/DHT11-Temperature%2FHumidity-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

Веб-сервер на ESP8266 для мониторинга температуры и влажности с датчика DHT11. Данные обновляются в реальном времени через веб-интерфейс.

## ✨ Возможности

- 📊 Мониторинг температуры (°C) и влажности (%)
- 🔄 Кеширование данных (хранение 4 секунды)
- 📱 Адаптивный дизайн для мобильных устройств
- 🔑 Авторизация (HTTP Basic Auth)
- 🔌 API для получения данных

## 🔧 Компоненты

| Компонент         | Назначение                     |
| ----------------- | ------------------------------ |
| ESP8266 (NodeMCU) | Микроконтроллер                |
| DHT11             | Датчик температуры и влажности |
| USB-кабель        | Питание и прошивка             |

## 🔌 Схема подключения

```
DHT11 → ESP8266
──────────────
VCC    → 3.3V
DATA   → GPIO5 (D1 на NodeMCU)
GND    → GND
```

## 📦 Установка

### Требования

- Arduino IDE с поддержкой ESP8266
- Установленные библиотеки:
  - `DHT sensor library`

## 🚀 Использование

1. **Подключение к ESP8266:**
   - После загрузки откройте Serial Monitor (115200 бод)
   - Узнайте IP-адрес, который получит ESP8266

2. **Откройте веб-интерфейс:**
   - В браузере перейдите по IP-адресу ESP8266
   - Пример: `http://192.168.1.100`

3. **Индикация (встроенный диод):**
   - Быстрое моргание - идёт подключение
   - Горит некоторое время, затем быстро мигает - подключение выполнено успешно, запуск сервера
   - Выключен (не горит) - подключение выполнено, сервер запущен

4. **Интерфейс:**
   - 🌡️ Температура в реальном времени
   - 💧 Влажность в реальном времени
   - 🕒 Время последнего обновления
   - 🔄 Кнопка ручного обновления

## 📡 API

### Получить данные

```http
GET /data
```

**Ответ:**

```json
{
  "t": 25.5, // температура (°C)
  "h": 60.3 // влажность (%)
}
```

### Получить веб-страницу

```http
GET /
```
